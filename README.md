# NJN-INTERNATIONAL-SDN-BHD

A modern and responsive **frontend portfolio website** developed for a company to showcase its services, projects, and brand identity.  
This website is designed with a clean layout, smooth interactions, and a professional look to create a strong online presence.


## Features

- Fully responsive design (desktop, tablet, mobile)  
- Clean and professional UI/UX  
- Company overview and services sections  
- Portfolio / projects showcase  
- Interactive elements using JavaScript  
- Smooth navigation and animations  


## Technologies Used

- **HTML5** – Website structure  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript (Vanilla JS)** – Interactivity and dynamic behavior
